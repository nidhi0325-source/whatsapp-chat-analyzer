# 📊 WhatsApp Chat Analyzer

An interactive Streamlit web app that analyzes WhatsApp chat data and generates meaningful insights.

## 🚀 Features

- 📌 Total messages & word count
- 📆 Monthly timeline analysis
- 🔥 Most active users
- 🗺️ Weekly activity heatmap
- ☁️ Word cloud visualization
- 😂 Emoji analysis
- 📅 Daily message frequency
- 📂 Support for uploaded chats
- 🧪 Built-in sample dataset

## 🛠 Tech Stack

- Python
- Pandas
- Matplotlib
- Seaborn
- WordCloud
- Streamlit

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
